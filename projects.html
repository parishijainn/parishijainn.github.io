Projects

Current Project

Gesture and Gaze Recognition for Accessible Collaborative Environments

This research project, part of the GRACE initiative at Carnegie Mellon, is focused on creating tools that combine gaze and gesture recognition to generate direct screen references. The goal is to make collaboration more accessible for blind and low-vision (BLV) users by translating non-verbal cues into accessible feedback. My work includes prototyping system components, running user studies, and analyzing multimodal interaction data.

Past Projects

King’s Closet

Demo | Code

A CMU Graphics–based recreation of Cher Horowitz’s virtual closet from *Clueless*. Users can browse, try on, and grade outfits using a combination of tops and bottoms. The game features an outfit grader powered by KMeans clustering for color-matching analysis, and a MediaPipe-powered hand-tracking mode for touchless navigation. I designed the grading system, hand-tracking mode, and user interface.

Stellar Sprint

Demo | Code

An engaging Flappy Bird–style game where players control a spaceship to collect stars and avoid obstacles. The game introduces increasing difficulty across multiple levels, with both keyboard and hand-tracking control modes. It also supports single-player, two-player, and player-vs-computer modes. I developed the gameplay logic, gesture control integration, and level progression system.

NeuralSeek Accessibility Agent

Demo

A multi-agent system built during my internship at NeuralSeek to automatically audit code, design docs, and UI screenshots for accessibility compliance (WCAG 2.1, ADA, AODA, Section 508). The tool detects high, medium, and low accessibility issues in minutes, tasks that typically take hours of manual review. It leverages OCR, routing logic, and modular agents with Virtual KBs and Guardrails. I designed workflows, tested agent modularity, and ensured scalability of accessibility audits.

Hack112: 112 Run

Code

A Temple Run–inspired game developed at Hack112 where players collect coins, dodge obstacles, and answer Python practice questions along the way. The game integrates computer vision, allowing users to control their character with finger-tracking gestures. I implemented the coin/obstacle mechanics and helped integrate the computer vision–based control system.

Duquesne Incline Website Redesign

Source Code

Our team was tasked with revamping the outdated Duquesne Incline website. We redesigned and developed a mobile-first, cross-browser compatible site using HTML, CSS, JavaScript, and jQuery. Key features include Google Maps API integration, form validation, and a dynamic image carousel. Through usability testing, we implemented improvements such as clearer navigation, sidebar consistency, and optimized content structure to enhance the user experience.
