Projects

Current Project

Gesture and Gaze Recognition for Accessible Collaborative Environments

As part of CMU’s GRACE initiative, I work on integrating gesture and gaze recognition to support blind and low-vision (BLV) users in collaborative spaces. My role includes prototyping accessibility features, conducting user studies, and analyzing multimodal interaction data to ensure interfaces are both functional and inclusive.

Past Projects

King’s Closet

Demo | Code

A recreation of Cher Horowitz’s virtual closet from Clueless. I built outfit grading with KMeans clustering to evaluate color harmony, and integrated gesture controls with MediaPipe for interactive outfit changes. This project combined playful design with technical rigor, showing how computer vision can enhance user experiences.

Stellar Sprint

Demo | Code

A Flappy Bird–style game with gesture control, scaling difficulty, and multiplayer support. I designed responsive mechanics and balanced gameplay loops, focusing on accessibility through intuitive controls.

NeuralSeek Accessibility Agent

Demo

A multi-agent system to audit websites and designs for accessibility compliance (WCAG/Section 508). I developed modular workflows for OCR parsing, code validation, and knowledge-based reasoning—transforming accessibility auditing into a faster, automated process.

Hack112: 112 Run

Code
