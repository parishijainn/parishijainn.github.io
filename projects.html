Projects

Current Project

Gesture and Gaze Recognition for Accessible Collaborative Environments

At CMU’s GRACE initiative, I contribute to research on gesture and gaze recognition that enables blind and low-vision (BLV) users to participate fully in collaborative environments. My responsibilities include designing interactive prototypes, running user studies with BLV participants, and analyzing multimodal interaction data to uncover how gaze and gesture can be leveraged as accessible input channels. This work highlights how inclusive design principles can scale to real-world environments.

Past Projects

King’s Closet

Demo | Code

A modern take on Cher Horowitz’s virtual closet from Clueless, this project blends computer vision, clustering, and interactive design. I developed a KMeans-based outfit grading system that analyzes color harmony and integrated gesture controls with MediaPipe to make browsing outfits feel seamless. The project challenged me to combine playful, pop-culture-inspired design with rigorous algorithms for real-time evaluation and interaction.

Stellar Sprint

Demo | Code

A gesture-controlled space adventure where players pilot a spacecraft through procedurally generated obstacles, adapting to increasing levels of difficulty. I designed dynamic physics-based movement, multiplayer functionality, and a progression system that kept gameplay engaging while ensuring accessibility through intuitive motion controls. This project demonstrated my ability to merge entertainment design with user-centered input systems.

NeuralSeek Accessibility Agent

Demo

A multi-agent system for automated accessibility audits across code, designs, and screenshots. I built modular agents for OCR, static code analysis, and validation against WCAG and Section 508 standards. A shared knowledge base allowed the system to learn from prior audits, making classification more reliable. The project transformed accessibility reviews from hours of manual effort to an automated, explainable workflow.

Hack112: 112 Run

Code

An educational game that combines learning and play. Players run through procedurally generated paths while solving Python problems, controlled with finger-tracking input. I focused on integrating educational content with engaging mechanics, demonstrating how gamification can motivate learning in a fun, interactive way.

Duquesne Incline Website Redesign

Source Code

Led a full redesign of the historic Duquesne Incline’s outdated website, emphasizing responsive design, accessibility, and usability. Added Google Maps integration, modern form validation, and tested the design with users to confirm that navigation and clarity were improved. This project showed me the impact of design updates on real community-facing services.
